# Статистический анализ вибрационных сигналов для диагностики подшипников качения

Курсовая работа по математической статистике: диагностика технического состояния подшипников качения по вибросигналам с помощью методов прикладной статистики. На реальных данных [CWRU Bearing Dataset](https://engineering.case.edu/bearingdatacenter) показано, что статистические характеристики вибрации позволяют объективно отличать исправный подшипник от дефектного и определять тип дефекта.

## О проекте

Вибрационная диагностика — один из основных методов предиктивного обслуживания вращающегося оборудования. В работе исследуются вибросигналы подшипников в четырёх состояниях и проверяется гипотеза о том, что переход от исправного состояния к дефектному меняет статистические свойства сигнала (разброс, форму распределения, «тяжесть хвостов»), а значит эти свойства можно использовать как диагностические признаки.

Анализ разбит на три раздела, повторяющих классическую логику статистического исследования: от разведочного описания данных — к оценкам и проверке гипотез — и далее к идентификации закона распределения.

## Данные

Используется **CWRU Bearing Dataset** (Case Western Reserve University), частота дискретизации 12 000 Гц, режим 1797 об/мин. Рассматриваются четыре класса сигналов, по ~121 800 отсчётов каждый:

| Класс | Описание |
|-------|----------|
| `Normal` | исправный подшипник без дефектов |
| `IR_14` | дефект внутренней обоймы (14 мил ≈ 0.36 мм) |
| `OR_14` | дефект внешней обоймы (14 мил) |
| `B_14` | дефект тела качения (14 мил) |

Каждый сигнал — одномерный временной ряд измерений ускорения вибрации (в единицах g). Данные подгружаются автоматически из открытого репозитория [`srigas/CWRU_Bearing_NumPy`](https://github.com/srigas/CWRU_Bearing_NumPy) при запуске ноутбука.

## Структура анализа

### Раздел 1. Предварительный статистический анализ
- Построение интервального вариационного ряда (число интервалов по формуле Стёрджеса).
- Гистограммы относительных частот и эмпирические функции распределения для всех классов.
- Расчёт выборочных характеристик: среднее, дисперсия, СКО, коэффициент вариации.
- Точечные оценки параметров тремя методами: **максимального правдоподобия**, **моментов** и **байесовским** (с сопряжённым нормально-обратногамма априором).
- Ключевой вывод раздела: превышение СКО дефектных сигналов над нормой как первичный диагностический признак.

### Раздел 2. Статистические оценки и проверка гипотез
- Точечные оценки: медиана, мода (через ядерную оценку плотности), среднее, дисперсия, СКО.
- **Доверительные интервалы** (γ = 0.95) для μ, σ², σ и медианы; для μ — на основе t-распределения, для σ² — на основе χ². 
- Проверка статистических гипотез:
  - **критерий Харке–Бера** — проверка нормальности;
  - **критерий Манна–Уитни** — сравнение распределений исправного и дефектных сигналов;
  - **критерий отношения правдоподобия (LRT)** — проверка равенства дисперсий.
- Визуализация: гистограммы с наложением ядерной оценки плотности и теоретической нормальной кривой.

### Раздел 3. Идентификация распределений
- **Критерий согласия Пирсона (χ²)** для двух гипотез о законе распределения: нормального и t-распределения Стьюдента (с объединением интервалов, где ожидаемые частоты малы).
- **Метод анаморфоз** — спрямление эмпирической функции распределения в координатах соответствующего закона с оценкой качества подгонки по R² и восстановлением параметров из коэффициентов регрессии.
- Сравнение двух методов идентификации и проверка их согласованности.

## Основные результаты

- **СКО** дефектных сигналов превышает норму в 2–3 раза — простой и устойчивый диагностический признак.
- **Коэффициент эксцесса** γ₂ > 10 характерен для дефектов внутренней обоймы и тела качения: дефекты порождают ударные импульсы, утяжеляющие хвосты распределения.
- **Форма распределения**: отклонение от нормальности в сторону лептокуртичности лучше описывается t-распределением Стьюдента, что согласуется с ударной природой дефектных сигналов.

Итоговый вывод: статистические методы дают объективную основу для диагностики состояния подшипников без привлечения сложных моделей — по набору интерпретируемых характеристик сигнала.
